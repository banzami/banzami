# Banzami Brand Assets

## Master Icon

`icon_master_1024.png` — 1024×1024 PNG, the single source of truth for all derived icons.

Do not edit this file directly. All resizes and format conversions should be derived from this master.

## Directory Structure

```
/branding
  icon_master_1024.png    ← master app icon (wine on white, no text)
  /social
    og_image_1200x630.png (to be generated from brand guidelines)
```

## Derivation Guide

| Platform | Size | Source |
|----------|------|--------|
| iOS App Store | 1024×1024 | icon_master_1024.png |
| Android Play Store | 512×512 | icon_master_1024.png |
| Android adaptive foreground | 108×108dp | icon_master_1024.png |
| Favicon (browser) | 32×32, 64×64 | icon_master_1024.png |
| Apple Touch Icon | 180×180 | icon_master_1024.png |
| Social Open Graph | 1200×630 | social/og_image_1200x630.png |
